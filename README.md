iGoTeK Neptune Prover v1.7 for HiveOS

Video Tutorial: https://youtu.be/AX4x20-7R-Y

A high-performance GPU Solo-mining software for Neptune Cash (NPT) cryptocurrency, optimized for NVIDIA GPUs with 21GB+ VRAM. I'm not the pool owner. I'm not the miner dev. All fees goes to the its developers.

Prerequisites  
HiveOS  
Neptune wallet address  
NVIDIA GPU with 21GB+ VRAM

Supported GPUs  
L40S  
RTX 6000 ADA  
A40  
RTX 6000 
RTX A5000 @ 220W ~6.50 MH/s (Core Clock Offset 200)  
RTX 5090 @ 580W ~39 MH/s  
RTX 5090 @ 450W ~30 MH/s  
RTX 4090 @ 450W ~21 MH/s  
RTX 4090 @ 280W ~16 MH/s     
RTX 3090 @ 320W ~9 MH/s (Core Clock Offset 150, Lock Core Clock 1740)
